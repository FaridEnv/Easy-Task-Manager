**Features**

\• Add new tasks  

\• View all tasks  

\• Mark tasks as done  

\• Edit tasks:

&nbsp; - Rename  

&nbsp; - Delete  

&nbsp; - Duplicate  

\• Tasks are stored in a JSON file (`tasks.json`) for persistence

\## Installation



1\. Install \[Python 3](https://www.python.org/downloads/) if not already installed.  

2\. Download or clone this repository:



```bash

git clone <https://github.com/FaridEnv/Easy-Task-Manager>

cd task_manager



---



\## How to Run



Execute the main script:



```bash

python task\_manager.py
