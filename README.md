### :rocket: 1. Features

• Add new tasks  

• View all tasks  

• Mark tasks as done  

• Edit tasks:

&nbsp; - Rename  

&nbsp; - Delete  

&nbsp; - Duplicate  

• Tasks are stored in a JSON file (`tasks.json`) for persistence  
 
---

### :gear: 2. Installation

1\. Install Python 3: https://www.python.org/downloads/ if not already installed.  

2\. Download or clone this repository:

```bash

git clone <https://github.com/FaridEnv/Easy-Task-Manager>

cd task_manager

```

---

### :arrow_forward: 3. How to Run

Execute the main script:
```bash

python task_manager.py

```

### :heavy_plus_sign: 4. Menu Options


### :computer: 5. Example Workflow

```bash
Run the application: python task\_manager.py
```
```bash
1. Select option 1 to add a task
2. Select option 2 to view all tasks
3. Select option 3 to mark a task as done
4. Select option 4 to edit a task
5. Use option 5 to Exit

Choose: 1

Enter task: organize all my references within Zotero database

Task added.

Choose: 2

Task List:

1. organize all my references within Zotero database - Not Done

```
